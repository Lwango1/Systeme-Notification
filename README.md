# Système de Notification Daniel Lwango

Ce projet permet de gérer des alertes réseau pour les services DG, Finance et Logistique.

## Contenu du dépôt
* **Master.py** : Code Python pour les postes clients.
* **Fichiers PHP** : Scripts pour le serveur (api_master.php, db_connect.php).
* **.gitignore** : Configuration pour exclure les fichiers inutiles.

## Configuration
Le serveur est configuré sur l'IP : **192.168.1.97**.
